Chipmunk v5.3.1 for PSP by Gefa
-----------------------------
This is a port of the chipmunk physics library v5.3.1 to PSP, implementing the oslib mod by sakya for graphic and input.
It also implements the cpMouse module by Tommy Thorsen

INSTALL
-------
To install Chipmunk v5.3.1 for PSP copy
libChipmunk.a in $PSPSDK/lib/

copy the chipmunk directory (containing header files) in $PSPSDK/include/
